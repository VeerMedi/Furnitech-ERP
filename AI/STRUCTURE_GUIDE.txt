================================================================================
                            WORKFLOW OPTIONS
================================================================================

OPTION 1: Manual Organization (You split data yourself)
--------------------------------------------------------
1. Annotate your images using Roboflow or LabelImg
2. Manually split 70/20/10 into train/val/test folders
3. Verify: python check_dataset.py
4. Train: python train_yolo_kfold.py


OPTION 2: Auto Split (Let script split for you)
------------------------------------------------
1. Put ALL images in:     raw_data/images/
2. Put ALL labels in:     raw_data/labels/
3. Run: python split_dataset.py --source raw_data
4. Verify: python check_dataset.py
5. Train: python train_yolo_kfold.py


OPTION 3: Use Roboflow Export (Recommended - Easiest!)
-------------------------------------------------------
1. Upload images to Roboflow.com
2. Annotate online (very easy UI)
3. Export as "YOLO v8" format
4. Download will already be split train/val/test!
5. Copy contents to: datasets/furniture_floorplan/
6. Verify: python check_dataset.py
7. Train: python train_yolo_kfold.py

================================================================================
                            LABEL FORMAT
================================================================================

Each .txt file contains one line per furniture item:

<class_id> <x_center> <y_center> <width> <height>

Example (floorplan_001.txt):
0 0.450 0.350 0.120 0.080    ← workstation at center (45%, 35%)
4 0.750 0.600 0.150 0.100    ← conference table
13 0.760 0.620 0.020 0.020   ← chair

All values are normalized (0-1):
- x_center: X position / image_width
- y_center: Y position / image_height
- width: Box width / image_width
- height: Box height / image_height
